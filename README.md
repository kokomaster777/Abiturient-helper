# Abiturient-helper
Чат-бот для помощи абитуриенту с поступлением
# Описание
Telegram-бот, который отвечает на вопросы абитуриента:
- о специальностях
- о поступлении
- о жизни в студенческом городке
- о стипендии и соц. гарантиях
